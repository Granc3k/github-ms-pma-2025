# Vánoční aplikace
- aplikace pro vytvoření dopisu Ježíškovi
- uživatel si do ní zapíše seznam dárků, co by chtěl
- zadá si jméno
- následně se automaticky vytvoří dopis s daným seznamem dárků, který si uživatel může zkopírovat do schránky

- navíc je zde featura darkmodu, kdyby někomu vadilo bílé pozadí
